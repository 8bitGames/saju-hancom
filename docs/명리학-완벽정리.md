## 1. 기본 구성요소

### 1.1 천간(天干) - 십천간

하늘의 기운을 나타내며, 정신적인 면을 상징합니다.

| 천간 | 한글 | 음양 | 오행 | 색상 |
|------|------|------|------|------|
| 甲 | 갑 | 양 | 목(木) | 청색 |
| 乙 | 을 | 음 | 목(木) | 청색 |
| 丙 | 병 | 양 | 화(火) | 적색 |
| 丁 | 정 | 음 | 화(火) | 적색 |
| 戊 | 무 | 양 | 토(土) | 황색 |
| 己 | 기 | 음 | 토(土) | 황색 |
| 庚 | 경 | 양 | 금(金) | 백색 |

### 1.3 육십갑자(六十甲子)

- 십간(10) × 십이지(12)의 조합에서 음양이 맞는 것만 조합
- 총 60개의 간지 조합 (갑자, 을축, 병인... 임술, 계해)
- 60년을 한 주기로 순환

### 1.4 지장간(支藏干)

- 지지 안에 숨어있는 천간
- 각 지지는 1~3개의 천간을 품고 있음
- 본기, 중기, 여기로 구분

## 2. 오행(五行)

### 2.1 오행의 기본 속성

| 오행 | 한자 | 방향 | 계절 | 색상 | 맛 | 장기 |
|------|------|------|------|------|-----|------|
| 목(木) | 나무 | 동 | 봄 | 청색 | 신맛 | 간 |
| 화(火) | 불 | 남 | 여름 | 적색 | 쓴맛 | 심장 |
| 토(土) | 흙 | 중앙 | 환절기 | 황색 | 단맛 | 비장 |
| 금(金) | 쇠 | 서 | 가을 | 백색 | 매운맛 | 폐 |
| 수(水) | 물 | 북 | 겨울 | 흑색 | 짠맛 | 신장 |

### 2.2 상생(相生) - 서로 돕는 관계

```
목(木) → 화(火) → 토(土) → 금(金) → 수(水) → 목(木)
  ↑_______________________________________________|
```

| 상생 | 설명 |
|------|------|
| 목생화(木生火) | 나무가 타서 불을 낳음 |
| 화생토(火生土) | 불이 재가 되어 흙을 낳음 |
| 토생금(土生金) | 흙에서 금속이 나옴 |
| 금생수(金生水) | 금속에서 물이 맺힘 (이슬) |
| 수생목(水生木) | 물이 나무를 살림 |

### 2.3 상극(相剋) - 서로 억제하는 관계

```
목 → 토 → 수 → 화 → 금 → 목
```

| 상극 | 설명 |
|------|------|
| 목극토(木剋土) | 나무가 흙에 뿌리를 내림 |
| 토극수(土剋水) | 흙이 물을 가둠 (제방) |
| 수극화(水剋火) | 물이 불을 끔 |
| 화극금(火剋金) | 불이 금을 녹임 |
| 금극목(金剋木) | 금(도끼)이 나무를 자름 |

### 2.4 상생상극의 조화

- 생(生)만으로는 생명력을 이어갈 수 없음
- 극(剋)이 있어야 균형과 긴장감이 생김
- 생과 극이 조화를 이루어야 건강한 사주

## 3. 행운론(行運論) - 운의 흐름

### 3.1 행운의 개념

**행운(行運)**: 외부에서 들어오는 변화 요인으로 사주를 자극하거나 관계를 맺어 변화시키는 것

| 구분 | 정의 | 특징 |
|------|------|------|
| 대운 | 사주가 만들어내는 기운 | 개인마다 다름 |
| 세운/월운/일운/시운 | 외부에서 유입하는 기운 | 모든 사람에게 동일 |

### 3.2 대운(大運) ⭐⭐⭐

**정의**: 사주 월주에서 발생하여 **10년마다** 바뀌는 큰 운의 흐름

#### 대운의 중요성
- 사주팔자를 **'자동차'**라고 하면, 대운은 그 차가 달리는 **'도로'**
- 사주가 좋아도 대운이 나쁘면 고생
- 사주가 나빠도 대운이 좋으면 만복

#### 순행/역행 판정

| 년간(양/음) | 남자 | 여자 |
|-------------|------|------|
| 양간(甲丙戊庚壬) | **순행** | 역행 |
| 음간(乙丁己辛癸) | 역행 | **순행** |

#### 대운수 계산법

1. **순행인 경우**: 출생일 → 다음 달 절기까지 날짜 ÷ 3
2. **역행인 경우**: 출생일 → 이전 달 절기까지 날짜 ÷ 3
3. **나머지 처리**:
   - 나머지 1: 버림
   - 나머지 2: +1

> ⚠️ 대운수는 **만 나이** 기준 (0세부터 시작)

#### 대운의 이중 작용
- **용(用) 작용**: 행운으로서 10년간 외부 변화 요인
- **체(體) 작용**: 부족한 오행을 보완하여 일시적으로 사주 본체처럼 기능

### 3.3 세운(歲運) / 연운(年運)

| 항목 | 내용 |
|------|------|
| 정의 | 1년을 주기로 변해가는 운 |
| 변경 기준 | **입춘** (양력 2월 4일경) |
| 천문학적 의미 | 지구 1공전 |
| 특징 | 모든 사람에게 동일한 간지 적용 |

#### 세운의 역할
- 대운으로 '시기의 도래' 파악
- 세운으로 '실행 여부의 적정성' 파악

### 3.4 월운(月運)

| 항목 | 내용 |
|------|------|
| 정의 | 한 달을 관장하는 운 |
| 변경 기준 | **12절기** |
| 천문학적 의미 | 지구 공전 각도 |
| 특징 | 사주원국과 직접적으로 작용 |

#### 12절기 기준

| 월 | 절기 | 월 | 절기 |
|-----|------|-----|------|
| 1월 | 입춘 | 7월 | 입추 |
| 2월 | 경칩 | 8월 | 백로 |
| 3월 | 청명 | 9월 | 한로 |
| 4월 | 입하 | 10월 | 입동 |
| 5월 | 망종 | 11월 | 대설 |
| 6월 | 소서 | 12월 | 소한 |

### 3.5 일운(日運) / 일진(日辰)

| 항목 | 내용 |
|------|------|
| 정의 | 하루를 관장하는 운 |
| 변경 기준 | **자시(子時)** - 밤 11시 30분 |
| 천문학적 의미 | 지구 1자전 |
| 일상 표현 | "오늘 일진이 좋다/안 좋다" |

### 3.6 시운(時運)

| 항목 | 내용 |
|------|------|
| 정의 | 2시간 단위의 운 |
| 변경 기준 | 2시간마다 |
| 천문학적 의미 | 지구 자전 각도 |

### 3.7 소운(小運)

- 대운이 시작되기 전 어린 시절의 운
- 주로 1세부터 대운 시작 전까지 적용
- 일부 학파에서만 사용
- 부모의 영향이 크기 때문에 중요도는 낮음

### 3.8 운의 해석 원칙

```
시간 → 일운 → 월운 → 세운 → 대운 → 사주
```

| 상위 운 | 하위 운 | 관계 |
|---------|---------|------|
| 대운 | 세운 | 시기 도래 → 실행 적정성 |
| 세운 | 월운 | 시기 도래 → 실행 적정성 |
| 월운 | 일운 | 시기 도래 → 실행 적정성 |
| 일운 | 시운 | 시기 도래 → 실행 적정성 |

## 4. 십성(十星) / 십신(十神)

### 4.1 기본 개념

- **기준**: 일간(日干) = 나 자신
- **원리**: 일간과 다른 글자들의 오행 관계 + 음양 관계

### 4.2 십성 구성

| 십성 | 한자 | 관계 | 음양 | 의미 |
|------|------|------|------|------|
| **비견(比肩)** | 比肩 | 같은 오행 | 같음 | 나와 동등한 존재, 형제 |
| **겁재(劫財)** | 劫財 | 같은 오행 | 다름 | 경쟁자, 라이벌 |
| **식신(食神)** | 食神 | 내가 생하는 | 같음 | 생활력, 재능, 표현력 |
| **상관(傷官)** | 傷官 | 내가 생하는 | 다름 | 반항심, 비판력 |
| **정재(正財)** | 正財 | 내가 극하는 | 다름 | 안정적 재물, 저축 |
| **편재(偏財)** | 偏財 | 내가 극하는 | 같음 | 유통 재물, 투자, 사업 |
| **정관(正官)** | 正官 | 나를 극하는 | 다름 | 안정된 직장, 명예 |
| **편관(偏官)** | 偏官 | 나를 극하는 | 같음 | 스트레스, 압박, 칠살 |
| **정인(正印)** | 正印 | 나를 생하는 | 다름 | 학문, 자격증, 어머니 |
| **편인(偏印)** | 偏印 | 나를 생하는 | 같음 | 특수기술, 계모 |

### 4.3 십성의 그룹화

| 그룹 | 구성 | 의미 |
|------|------|------|
| 비겁(比劫) | 비견 + 겁재 | 나와 같은 오행 (자아) |
| 식상(食傷) | 식신 + 상관 | 내가 생하는 오행 (표현) |
| 재성(財星) | 정재 + 편재 | 내가 극하는 오행 (재물) |
| 관성(官星) | 정관 + 편관 | 나를 극하는 오행 (직장/명예) |
| 인성(印星) | 정인 + 편인 | 나를 생하는 오행 (학문/보호) |

### 4.4 길신과 흉신

| 구분 | 십성 | 설명 |
|------|------|------|
| **사길신** | 재성, 정관, 정인, 식신 | 일반적으로 좋은 기운 |
| **사흉신** | 편관(칠살), 상관, 겁재, 편인(효신) | 주의가 필요한 기운 |

> ⚠️ 흉신도 잘 활용하면 좋은 결과를 낳을 수 있음

### 4.5 십성의 상생 흐름

**정(正)의 흐름** - 생(生)을 좋아함:
```
비견 → 식신 → 정재 → 정관 → 정인 → 비견
```

**편(偏)의 흐름** - 극(剋)을 받아야 좋음:
```
겁재, 상관, 편재, 편관, 편인
```

## 5. 용신(用神) 체계

### 5.1 용신의 개념

사주팔자에서 약점이나 부족한 점을 보완할 수 있는 핵심 오행

### 5.2 용신의 종류

| 용신 유형 | 설명 |
|-----------|------|
| **억부용신** | 신강하면 억제, 신약하면 부조 |
| **조후용신** | 계절의 한난조습을 조절 |
| **통관용신** | 상극 관계를 통하게 함 |
| **병약용신** | 사주의 병을 치료하는 약 |
| **격국용신** | 격국을 완성하는 용신 |

### 5.3 오신(五神) 체계

| 신(神) | 한자 | 역할 | 관계 |
|--------|------|------|------|
| **용신(用神)** | 用神 | 가장 필요한 오행 | 핵심 |
| **희신(喜神)** | 喜神 | 용신을 돕는 오행 | 용신을 생(生) |
| **기신(忌神)** | 忌神 | 용신을 방해하는 오행 | 용신을 극(剋) |
| **구신(仇神)** | 仇神 | 기신을 돕는 오행 | 기신을 생(生) |
| **한신(閑神)** | 閑神 | 나머지 오행 | 상황에 따라 변함 |

### 5.4 용신 예시

**목(木)이 용신인 경우:**

| 오행 | 역할 | 이유 |
|------|------|------|
| 목(木) | 용신 | 필요한 오행 |
| 수(水) | 희신 | 수생목 (용신을 생) |
| 금(金) | 기신 | 금극목 (용신을 극) |
| 토(土) | 구신 | 토생금 (기신을 생) |
| 화(火) | 한신 | 직접적 관계 없음 |

## 6. 합충형파해(合沖刑破害)

### 6.1 천간합(天干合)

양천간과 음천간이 만나 다른 오행으로 변화

| 합 | 조합 | 결과 오행 |
|-----|------|----------|
| 갑기합 | 甲 + 己 | 토(土) |
| 을경합 | 乙 + 庚 | 금(金) |
| 병신합 | 丙 + 辛 | 수(水) |
| 정임합 | 丁 + 壬 | 목(木) |
| 무계합 | 戊 + 癸 | 화(火) |

### 6.2 지지합

#### 삼합(三合)
3글자가 모여 하나의 오행으로 변화 (가장 강력)

| 삼합 | 구성 | 결과 |
|------|------|------|
| 인오술(寅午戌) | 생지+왕지+묘지 | 화(火)국 |
| 신자진(申子辰) | 생지+왕지+묘지 | 수(水)국 |
| 사유축(巳酉丑) | 생지+왕지+묘지 | 금(金)국 |
| 해묘미(亥卯未) | 생지+왕지+묘지 | 목(木)국 |

#### 방합(方合)
같은 방향의 3글자가 모임

| 방합 | 구성 | 결과 |
|------|------|------|
| 인묘진(寅卯辰) | 동방 | 목(木)국 |
| 사오미(巳午未) | 남방 | 화(火)국 |
| 신유술(申酉戌) | 서방 | 금(金)국 |
| 해자축(亥子丑) | 북방 | 수(水)국 |

#### 육합(六合)
2글자씩 합 (삼합보다 약함)

| 육합 | 결과 |
|------|------|
| 자축합(子丑) | 토(土) |
| 인해합(寅亥) | 목(木) |
| 묘술합(卯戌) | 화(火) |
| 진유합(辰酉) | 금(金) |
| 사신합(巳申) | 수(水) |
| 오미합(午未) | 화(火) |

#### 합의 강도
```
삼합 > 방합 > 육합 > 반합 > 암합
```

### 6.3 충(沖) - 육충(六沖)

정반대 방위의 글자가 충돌 (7번째 자리)

| 충 | 글자 | 방향 |
|-----|------|------|
| 자오충 | 子 ↔ 午 | 북 ↔ 남 |
| 축미충 | 丑 ↔ 未 | 토끼리 (붕충) |
| 인신충 | 寅 ↔ 申 | 동북 ↔ 서남 |
| 묘유충 | 卯 ↔ 酉 | 동 ↔ 서 |
| 진술충 | 辰 ↔ 戌 | 토끼리 (붕충) |
| 사해충 | 巳 ↔ 亥 | 동남 ↔ 서북 |

#### 충의 해석
- 충 = 무조건 흉이 아님
- 나쁜 기운을 충으로 깨뜨리면 오히려 좋음
- 에너지의 흐름과 변화를 의미

### 6.4 형(刑)

형벌, 살상, 제재를 의미

| 형 종류 | 구성 | 의미 |
|---------|------|------|
| 삼형살 | 인사신(寅巳申) | 무은지형 |
| 삼형살 | 축술미(丑戌未) | 지세지형 |
| 자형 | 진진, 오오, 유유, 해해 | 자기 자신을 해침 |
| 상형 | 자묘(子卯) | 무례지형 |

#### 형의 영향
- 사고, 소송, 질병, 수술
- 급작스런 중단
- 합이 되어있어도 형이 발생

### 6.5 파(破)

합을 깨뜨리는 작용

| 파 | 관계 |
|-----|------|
| 자유파 | 子 - 酉 |
| 묘오파 | 卯 - 午 |
| 인해파 | 寅 - 亥 |
| 사신파 | 巳 - 申 |
| 축진파 | 丑 - 辰 |
| 술미파 | 戌 - 未 |

### 6.6 해(害)

육합을 방해하는 작용

| 해 | 관계 | 설명 |
|-----|------|------|
| 자미해 | 子 - 未 | 자축합을 미가 방해 |
| 축오해 | 丑 - 午 | 오미합을 축이 방해 |
| 인사해 | 寅 - 巳 | 인해합을 사가 방해 |
| 묘진해 | 卯 - 辰 | 묘술합을 진이 방해 |
| 신해해 | 申 - 亥 | 사신합을 해가 방해 |
| 유술해 | 酉 - 戌 | 진유합을 술이 방해 |

## 7. 격국(格局)

### 7.1 격국의 개념

사주의 구조와 품격을 나타내는 분류 체계

- **일간(日干)**을 기준으로 **월지(月支)**를 중심으로 판단
- 격국에 따라 용신과 해석 방향이 달라짐

### 7.2 정격(正格) - 10가지

일반적인 사주에 적용되는 격

| 격 | 설명 |
|-----|------|
| 식신격 | 월지에서 식신이 투출 |
| 상관격 | 월지에서 상관이 투출 |
| 정재격 | 월지에서 정재가 투출 |
| 편재격 | 월지에서 편재가 투출 |
| 정관격 | 월지에서 정관이 투출 |
| 편관격 | 월지에서 편관이 투출 |
| 정인격 | 월지에서 정인이 투출 |
| 편인격 | 월지에서 편인이 투출 |
| 비견격 | 월지에서 비견이 투출 |
| 겁재격 | 월지에서 겁재가 투출 |

### 7.3 외격(外格) - 특수한 사주

일반적이지 않은 특수한 구조

#### 종격(從格)
특정 세력이 너무 강해 그것을 따라가는 격

| 종격 | 따르는 대상 | 용신 |
|------|-------------|------|
| 종아격(從兒格) | 식상 | 식신/상관 |
| 종재격(從財格) | 재성 | 정재/편재 |
| 종살격(從殺格) | 관성 | 정관/편관 |
| 종강격(從强格) | 비겁+인성 | 비견/인성 |

#### 화격(化格)
천간합으로 변하는 오행을 따르는 격

| 화격 | 천간합 | 화하는 오행 |
|------|--------|-------------|
| 화토격 | 갑기합 | 토(土) |
| 화금격 | 을경합 | 금(金) |
| 화수격 | 병신합 | 수(水) |
| 화목격 | 정임합 | 목(木) |
| 화화격 | 무계합 | 화(火) |

#### 기타 외격
- **전왕격**: 한 오행이 압도적으로 강함
- **일행득기격**: 특정 오행으로 통일
- **양신성상격**: 두 오행이 균형
- **삼기성상격**: 세 오행이 균형

### 7.4 진격과 가격

| 구분 | 설명 |
|------|------|
| 진격(眞格) | 순수하게 격이 성립 |
| 가격(假格) | 방해 요소가 있어 불완전 |

## 8. 신살(神煞)

### 8.1 신살의 개념

사주팔자에서 특수한 관계를 나타내는 기운

- **신(神)**: 길신(吉神) - 좋은 기운
- **살(煞)**: 흉신(凶神) - 나쁜 기운

### 8.2 주요 신살

#### 도화살(桃花殺) 🌸
| 항목 | 내용 |
|------|------|
| 의미 | 이성관계, 매력, 인기 |
| 전통 해석 | 복잡한 이성문제 |
| 현대 해석 | 연예계 적성, 대인관계 우수 |
| 판별 | 일지/년지 기준 → 자오묘유에서 확인 |

#### 역마살(驛馬殺) 🐎
| 항목 | 내용 |
|------|------|
| 의미 | 이동, 활동, 통신, 전파 |
| 전통 해석 | 한 곳에 정착 못함 (흉) |
| 현대 해석 | 해외 활동, 무역, 여행에 유리 |
| 판별 | 인신사해(생지)에서 확인 |

#### 화개살(華蓋殺) 📚
| 항목 | 내용 |
|------|------|
| 의미 | 종교, 학문, 예술 |
| 특징 | 고독하지만 깊은 사고 |
| 현대 해석 | 철학적 성향, 예술가 기질 |

#### 괴강살(魁罡殺) ⚔️
| 항목 | 내용 |
|------|------|
| 의미 | 강한 결단력, 권력 |
| 해당 일주 | 경진, 임진, 경술, 임술 |
| 특징 | 맺고 끊음이 확실, 용기 |

#### 귀문관살(鬼門關殺) 👁️
| 항목 | 내용 |
|------|------|
| 의미 | 신비한 기운, 통찰력 |
| 전통 해석 | 귀신이 붙기 쉬움 |
| 현대 해석 | 매우 영리함, 직감력 |

#### 양인살(羊刃殺) 🗡️
| 항목 | 내용 |
|------|------|
| 의미 | 날카로움, 결단력 |
| 특징 | 극단적인 성향 |
| 현대 해석 | 전문직(의사, 판사 등) 적성 |

#### 백호대살(白虎大煞) 🐅
| 항목 | 내용 |
|------|------|
| 의미 | 피를 보는 살 |
| 주의 | 사고, 수술 |
| 현대 해석 | 의료계, 정육업 등 적성 |

### 8.3 현대적 신살 해석

> ⚠️ 시대에 따라 신살 해석이 달라짐

| 전통 | 현대 |
|------|------|
| 도화살 = 흉 | 인기, 연예계 적성 |
| 역마살 = 흉 | 해외, 무역, 여행 유리 |
| 화개살 = 고독 | 학문, 예술 적성 |

## 9. 영상 핵심 내용 - 강헌 명리학자

### 9.1 영상 정보

- **채널**: 지식인사이드
- **링크**: https://www.youtube.com/watch?v=4LZd2T-YxjI

### 9.2 강헌 명리학자 소개

| 항목 | 내용 |
|------|------|
| 이름 | 강헌 (姜憲) |
| 출생 | 1962년 |
| 전직 | 대중음악 평론가, 영화/공연 제작자 |
| 현직 | 명리학자 |
| 저서 | 《전복과 반전의 순간》, 《명리-운명을 읽다》 |

#### 명리학 입문 계기
- 43세에 갑자기 의식을 잃고 쓰러짐
- 깨어난 후 자신의 운명이 궁금해짐
- 이후 명리학에 몰두하여 전문가가 됨

### 9.3 핵심 메시지

#### 1. 같은 사주, 다른 인생

**Q: 같은 날 태어난 사람들은 왜 다른 인생을 살까?**

| 요인 | 설명 |
|------|------|
| 시대 배경 | 태어난 시대의 사회적 환경 |
| 지역 환경 | 내륙/해안, 한국/외국 등 |
| 가정 환경 | 유복/가난, 부모의 성향 |
| 집안 내력 | 가문의 역사와 문화 |
| 배우자 | 결혼 상대의 영향 |

#### 2. 사주와 대운의 비유

```
사주팔자 = 자동차 (타고난 것)
대운 = 도로 (시기적 환경)
```

| 사주 | 대운 | 결과 |
|------|------|------|
| 좋음 | 나쁨 | 고생 |
| 나쁨 | 좋음 | 만복 |
| 좋음 | 좋음 | 대성공 |
| 나쁨 | 나쁨 | 어려움 |

#### 3. 사주 운용의 중요성

> "사주팔자는 그릇이고, 그 그릇을 어떻게 쓸 것인가 하는 개개인의 '마음'에 훨씬 더 방점이 찍혀있다"

| 사주 | 운용 | 결과 |
|------|------|------|
| 좋음 | 잘못 운용 | 거지, 급살 가능 |
| 나쁨 | 세심히 운용 | 본인 역량 이상 성취 |

#### 4. 명리학의 본질

- 명리학은 **점술이 아닌 동양철학**
- 자연의 이치를 통해 '나 자신'을 이해
- 운명을 예측하는 것이 아닌, **운명을 운용하는 지혜**
