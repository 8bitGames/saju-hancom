## 1. 핵심 철학: 청리움의 기운으로 사주를 읽다

### 1.1 The Core Concept

**청리움 (淸rium)** = "맑은 정기가 모이는 곳"

청기운은 단순한 사주 앱이 아닙니다. **청리움의 명당 기운을 디지털로 전달하는 영적 채널**입니다.

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   청리움 (물리적 명당)                                        │
│        ↓                                                    │
│   맑은 정기 (淸氣)                                           │
│        ↓                                                    │
│   오방신 (五方神) - 다섯 수호신                               │
│        ↓                                                    │
│   청기운 앱 - 디지털 채널                                     │
│        ↓                                                    │
│   사용자에게 명당의 지혜 전달                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**핵심 메시지**: 청기운의 모든 사주 해석은 청리움 명당에서 흘러나오는 맑은 기운에 기반합니다.

### 1.2 Subliminal Integration Principles

| 원칙 | 설명 | 예시 |
|-----|------|-----|
| **스며들기** | 청리움을 직접 언급하지 않고 개념을 녹여넣기 | "명당의 기운이 전합니다" |
| **자연스러움** | 프로모션이 아닌 세계관의 일부로 느껴지게 | 수호신이 청리움에 거주하는 설정 |
| **호기심 유발** | 사용자가 스스로 청리움을 찾아보게 만들기 | "청룡이 지키는 녹차밭에서..." |
| **가치 우선** | 청리움 노출 없이도 완전한 경험 제공 | 사주 해석 자체가 훌륭해야 함 |

## 2. 세계관 구축: 오방신과 청리움

### 2.1 The Lore (스토리 설정)

> 보리산 자락, 맑은 정기가 모이는 청리움에는 다섯 수호신이 깃들어 있습니다.
> 이들은 각자의 성지에서 인간 세상을 지켜보며, 운명의 길을 안내합니다.
> 청기운은 이 수호신들의 지혜를 당신에게 전달하는 영적 통로입니다.

### 2.2 Five Guardians & Their Domains

| 수호신 | 오행 | 청리움 성지 | 역할 | 향기 |
|-------|-----|-----------|-----|-----|
| **청룡** | 木 (목) | 녹차밭 | 성장의 인도자 | 소나무, 녹차, 이끼 |
| **주작** | 火 (화) | 용소 | 열정의 불꽃 | 계피, 생강, 정향 |
| **황룡** | 土 (토) | 약초원 | 대지의 양육자 | 백단, 침향, 꿀 |
| **백호** | 金 (금) | 오하산방 | 절제의 스승 | 유향, 매화, 백합 |
| **현무** | 水 (수) | 명당 기도터 | 신비의 수호자 | 연꽃, 해초, 오미자 |

### 2.3 How It Manifests in the App

**사주 분석 결과에서:**
```
"당신의 사주를 분석한 결과, 목(木) 기운이 가장 강합니다.

청룡은 녹차밭의 푸른 기운 속에서 당신의 성장을 지켜봅니다.
새벽 안개가 피어오르는 차밭에서, 청룡이 전하는 메시지입니다..."
```

**오늘의 운세에서:**
```
"오늘 아침, 약초원에서 황룡이 전해온 기운입니다.
```

## 4. 서브리미널 터치포인트 맵

### 4.1 Direct vs. Indirect Exposure

| Level | 노출 방식 | 예시 | 빈도 |
|-------|----------|-----|-----|
| **L1: 완전 간접** | 청리움 언급 없이 개념만 | "명당의 기운", "보리산 자락" | 매일 |
| **L2: 세계관 내** | 수호신 성지로 장소 언급 | "청룡이 지키는 녹차밭에서..." | 자주 |
| **L3: 호기심 유발** | 장소/경험 암시 | "실제로 이 기운을 느낄 수 있는 곳이..." | 가끔 |
| **L4: 명시적 링크** | 청리움 직접 언급 (선택적) | "청리움 알아보기 →" | 드물게 |

### 4.2 Touchpoint Distribution

```
├── L1: "오늘 명당에서 흘러온 기운은..."

└── L2: "황룡이 약초원에서 전합니다"

├── L2: "청룡은 녹차밭에서 당신을 지켜봅니다"
```
